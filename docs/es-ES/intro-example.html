Ejemplo de introducción

Antes de empezar veamos una pequeña muestra del la API de ReasonReact. No te preocupes si hay cosas que no entiendes, la documentación las cubrirá todas.

let component = ReasonReact.statelessComponent("Greeting");

/* guión bajo antes de un nombre de variable significa que no será usada. We name them for clarity */
let make = (~name, _children) => {
  ...component,
  render: (_self) => <button> {ReasonReact.stringToElement("Hola!")} </button>
};
