Installation

Aviso: Para a configuração inicial do editor Reason + BuckleScript, clique aqui.

To easily try ReasonReact, we offer two solutions.

Bsb

Nossa opção preferida na maioria dos casos. Caso seja sua primeira vez usando Reason React, sinta-se livre em usar a opção mais familiar para criar um aplicativo react, veja abaixo:

npm install -g bs-platform
bsb -init my-react-app -theme react
cd my-react-app && npm install && npm start
# in another tab
npm run webpack

O sistema de compilação do BuckleScript bsb possui um comando init que gera um projeto automaticamente para você. O tema react oferece uma solução leve, otimizada para fácil aprendizagem e com grande facilidade de integração em um projeto existente.

It compiles to straightforward JS files, so you can open index.html directly from the file system. No server needed.

Reason Scripts (Aka Create-React-App)

Reason-scripts fornece uma experiência familiar aos usuários de ReactJS que já estão familiarizados com create-react-app. É uma solução abrangente. No entanto, caso seja pesado demais para seu gosto, tente a primeira opção acima (bsb).

Como o create-react-app, o reason-scripts vem com um servidor e hotloading incluso.
